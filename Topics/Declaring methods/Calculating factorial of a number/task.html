Calculating factorial of a number

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You are given the task to calculate factorial of a number. Write a Java program where you declare a method that takes an integer as input (ranging from 1 to 12) and returns the factorial of that number. Factorial of a number is the product of all positive integers less than or equal to that number. For example, factorial of 5 is 5*4*3*2*1 = 120.
Sample Input:
5

Sample Output:
120


Sample Input:
3

Sample Output:
6



Memory limit: 256 MB
Time limit: 5 seconds
